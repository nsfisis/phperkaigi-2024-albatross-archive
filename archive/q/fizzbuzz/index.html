FizzBuzz

いわゆる「FizzBuzz」を実装してください。 1から100までの数字について、その数が3の倍数なら「Fizz」、5の倍数なら「Buzz」、15の倍数なら「FizzBuzz」、それ以外ならその数そのものを出力してください。それぞれの出力は改行で区切ってください。

実装例

for ($i = 1; $i <= 100; $i++) {
  echo match ([$i % 3 === 0, $i % 5 === 0]) {
    [true, true] => "FizzBuzz",
    [true, false] => "Fizz",
    [false, true] => "Buzz",
    [false, false] => $i,
  }, PHP_EOL;
}
